# Changelog

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-07-10

### Added

- Per-turn semantic model and reasoning routing for Codex CLI.
- Local loopback proxy with per-process bearer-token protection.
- Schema-validated classifier decisions, explicit route overrides, and model-catalog fallback resolution.
- Privacy-aware audit metadata and a local dashboard for routed cost, baseline cost, savings, usage, and route distribution.
- OpenAI Responses API and native Codex classifier backends.

### Known limitations

- Codex App Server integration and model catalogs evolve independently of this project.
- Cost figures are estimates based on configured token pricing; they are not provider invoices.
